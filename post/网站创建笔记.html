以下全部代码均使用GIT BASH

1. 下载 Git

https://git-scm.com/

2. 下载 Node.js

https://nodejs.org/zh-cn/ 长期维护版

3. 验证上面是否成功安装

node -v

npm -v

4. Hexo 网站

https://hexo.io/zh-cn/

5. Hexo 流程

安装

npm install hexo-cli -g

验证.

hexo -v

当前位置

pwd

本地部署

hexo s

初始化

hexo g

本地预览

hexo server

6. Aurora

1. 安装

npm install hexo-theme-aurora –save

2. 复制配置文件

  1. 复制 子目录 node_modules ->hexo-theme-aurora->_config.yml 文件内容
  2. 黏贴 到根目录 _config.aurora.yml 文件中

3. 修改配置文件

打开在 Hexo 根目录下的 _config.yml
把 permalink 参数为 /post/:title.html
把 highlight 的启用改为false
把 prismjs 的启用改为true
把 prismjs 下的 preprocess 改为 false
把 theme 修改为 aurora

7.创建About页面

主题默认开启了about页面，所以我们需要创建这个 about 页，要不主题就无法正常显示 about 页面，只需要执行这个 Hexo 的命令：

hexo new page about

执行完毕后，你会发现在 source/ 文件中多处了一个新的文件夹：

你可以随意修改这个 index.md markdown 文件中的内容，此文件的内容将会现在在我们的 about 页面中。

8.连接 GITHUB

1. 随便一个文件夹

打开Gitbash 输入ssh

2. 获取ssh 钥匙

ssh-keygen -t rsa -C “你的邮箱”

3. 打开文件复制内容

C:\Users\username.ssh\id_rsa.pub

4. 黏贴上述代码到

Github settiong ssh设置里

5. 确认连接

ssh -T git@github.com

6. 修改配置文件 打开根目录 _config.yml

deploy:
type: git
repo: https://github.com/XXXXXX/XXXXXXX.github.io
branch: main

7. 自动部署发布工具

npm install hexo-deployer-git –save

8. 上传内容

hexo d

9. 基于已经安装好个人hexo博客，已经上传过文章，后续上传文章时，按照如下步骤：

  1. 打开Blog/source/_posts文件夹
  2. 将新的.md文件存放进来
  3. 右键，Git Bash Here
  4. 输入：hexo g，回车
  5. 输入：hexo d，回车
  6. 到这里，可以静置等待上传完成。
